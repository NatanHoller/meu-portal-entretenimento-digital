# &#127916; Meu Portal de Entretenimento Digital

Projeto desenvolvido para a disciplina de Desenvolvimento de Sistemas utilizando HTML e CSS.

## &#128204; Sobre o projeto

O site apresenta um portal temático de entretenimento digital dividido em três seções principais:

- &#127909; Universos do Cinema e Séries
- &#127911; Ritmos, Podcasts e Cultura Digital
- &#127918; Games, Inovação e Futuro Tecnológico

O objetivo da atividade foi praticar conceitos de:

- Estrutura HTML
- CSS Inline, Interno e Externo
- Classes e IDs
- Flexbox
- Responsividade
- Organização visual de páginas web
